## 📦 Installation

### Prerequisites

- **Node.js** v16+ or **Python** 3.9+
- **Google Gemini API Key** (free at [ai.google.dev](https://ai.google.dev))
- **Groq API Key** (free at [console.groq.com](https://console.groq.com/keys))
- Modern web browser (Chrome, Firefox, Safari, Edge)

### Step-by-Step Setup

#### 1. Clone Repository
```bash
git clone https://github.com/yourusername/novaagent.git
cd novaagent
```

#### 2. Install Dependencies
```bash
npm install
# OR if using Python
pip install -r requirements.txt
```

#### 3. Configure API Keys
Create `.env` file:
```env
# Google Gemini
GEMINI_API_KEY=your_gemini_api_key_here

# Groq (Optional but recommended)
GROQ_API_KEY=your_groq_api_key_here

# App Configuration
PORT=8501
NODE_ENV=development
LOG_LEVEL=debug
```

#### 4. Run Application
```bash
# Development
npm run dev

# Production
npm run build && npm start
```

#### 5. Access Dashboard
Open browser: `http://localhost:8501`

## 🔌 API Reference

### Authentication
All requests require proper API key configuration in settings.

### Endpoints

#### Initialize Agent
```bash
POST /api/agents/initialize

Request:
{
  "agentName": "Sam Altman",
  "apiKey": "your_gemini_key"
}

Response:
{
  "status": "success",
  "agentId": "sam_altman",
  "message": "Neural link established"
}
```

#### Send Message
```bash
POST /api/chat/message

Request:
{
  "agentId": "sam_altman",
  "message": "What is your AGI strategy?",
  "conversationId": "conv_123"
}

Response:
{
  "agentResponse": "...",
  "latency": "14.2ms",
  "tokens": 245,
  "ragSources": ["source1", "source2"]
}
```

#### Get Agent Status
```bash
GET /api/agents/{agentId}/status

Response:
{
  "agentId": "sam_altman",
  "status": "online",
  "ping": "14ms",
  "activeUsers": 3,
  "lastActivity": "2 minutes ago"
}
```

#### List All Agents
```bash
GET /api/agents

Response:
{
  "agents": [
    {
      "id": "sam_altman",
      "name": "Sam Altman",
      "status": "online",
      "capabilities": [...]
    },
    ...
  ]
}
```
